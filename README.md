# Personal Portfolio Website

A modern and responsive portfolio website built to showcase my skills, projects, and achievements as a web developer. The website serves as a personal brand and provides visitors with information about me, my work, and ways to get in touch.

## Features

* Responsive design for all devices
* Clean and modern user interface
* About Me section
* Skills section
* Projects showcase
* Contact section
* Smooth navigation
* Optimized layout and performance

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to build a professional online presence and demonstrate front-end web development skills through a real-world portfolio website.
